## เกี่ยวกับหนังสือเล่มนี้

เฟรมเวิร์ก AI ในปัจจุบันเหมาะอย่างยิ่งสำหรับการสร้างโปรโตไทป์ แต่เมื่อเอเจนต์อัตโนมัติต้องทำงานอย่างน่าเชื่อถือและต่อเนื่องในระบบการผลิต นักพัฒนาต้องจัดการทุกอย่างด้วยตัวเอง

**หนังสือเล่มนี้เชื่อมช่องว่างที่สำคัญระหว่างโปรโตไทป์และระบบการผลิต**

ในขณะที่แหล่งข้อมูลหลายแห่งมุ่งเน้นการสร้างแชทบอทหรือการทำงานอัตโนมัติอย่างง่าย หนังสือเล่มนี้จัดการกับปัญหาที่ยากซึ่งเกิดขึ้นเมื่อเอเจนต์ต้องทำงานอย่างอัตโนมัติและต่อเนื่องในระดับขนาดใหญ่

### หนังสือเล่มนี้เหมาะสำหรับใคร

หนังสือเล่มนี้เขียนขึ้นสำหรับ:

- ผู้นำทางเทคนิคที่รับผิดชอบการตัดสินใจด้านสถาปัตยกรรม
- วิศวกรซอฟต์แวร์ที่ออกแบบแอปพลิเคชันแบบเอเจนต์ที่เชื่อถือได้และขยายขนาดได้

หนังสือเล่มนี้สันนิษฐานว่าคุณมี:

- ประสบการณ์การเขียนโปรแกรมในภาษาสมัยใหม่อย่างน้อยหนึ่งภาษา (TypeScript, Python, ฯลฯ)
- ความเข้าใจพื้นฐานเกี่ยวกับ API ระบบกระจาย และแนวคิดฐานข้อมูล
- ความเข้าใจพื้นฐานเกี่ยวกับ AI และ LLMs

### แนวทางและปรัชญา

หนังสือเล่มนี้ใช้แนวทางตามหลักการเบื้องต้นสำหรับระบบเอเจนต์ แทนที่จะส่งเสริมเฟรมเวิร์กในปัจจุบันที่จะล้าสมัยในวันพรุ่งนี้ เรามุ่งเน้นที่รูปแบบและหลักการพื้นฐานที่จะคงอยู่นานกว่าการนำไปใช้งานเฉพาะใดๆ

ทุกแนวคิดเปลี่ยนจากทฤษฎีสู่การปฏิบัติ นำเสนอโมเดลทางความคิดที่เชื่อถือได้ พร้อมทั้งตัวอย่างโค้ดที่สมจริงเพื่อสร้างความเข้าใจแบบองค์รวมเกี่ยวกับระบบเอเจนต์

## เกี่ยวกับผู้เขียน

**โดมินิค ทอร์โนว์** เป็นผู้ก่อตั้งและซีอีโอของ Resonate HQ, Inc. เขาศึกษาวิศวกรรมระบบซอฟต์แวร์ที่สถาบัน Hasso Plattner ในเมืองพอทสดัม ประเทศเยอรมนี และนำประสบการณ์ในอุตสาหกรรมมากกว่าสองทศวรรษมาสู่งานของเขา

โดมินิคเชี่ยวชาญด้านวิศวกรรมระบบ การคิดเชิงระบบ และการสร้างแบบจำลองระบบ โดยเน้นเทคนิคทางการและแนวคิดที่ทำให้ระบบคู่ขนานและกระจายถูกต้องตั้งแต่การสร้าง งานของเขาเชื่อมช่องว่างระหว่างวิทยาการคอมพิวเตอร์เชิงทฤษฎีและวิศวกรรมซอฟต์แวร์เชิงปฏิบัติ ทำให้แนวคิดที่ซับซ้อนเข้าถึงได้สำหรับผู้ปฏิบัติงาน

โดมินิคเป็นผู้เขียนหนังสือ **[Think Distributed Systems](https://www.manning.com/books/think-distributed-systems)** (Manning Publications) และเป็นวิทยากรประจำในการประชุมอุตสาหกรรมชั้นนำ รวมถึง Systems Distributed, Strange Loop และ KubeCon
